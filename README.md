# AI-Enhanced Interactivity in eLearning

This project is a microlearning module designed to explore how artificial intelligence can enhance interaction in online learning environments.

## Overview
This course was created for graduate students in Learning Design and Technology. It examines different types of learner interaction and demonstrates how AI tools can support engagement in digital learning environments.

## Learning Objectives
- Define interactivity in eLearning
- Identify AI strategies that enhance learner interaction
- Evaluate ethical considerations when using AI in learning design

## Tools Used
- Articulate Rise 360
- AI research tools
- Instructional design frameworks
